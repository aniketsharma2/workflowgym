## 🚀 What is GMS?

GMS (Gym Management Software) is a cloud-based SaaS platform that helps **gym owners across India** manage everything from one dashboard — members, attendance, payments, staff, workout plans, and WhatsApp automation.

**Trusted by 10,000+ gyms | 500,000+ members managed | 4.8/5 rating**

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 👥 **Member Management** | Full profiles, QR membership cards, plan history |
| 📱 **QR Code Attendance** | Scan to check in — kiosk mode, GPS check-in |
| 💬 **WhatsApp Automation** | Auto-send fee reminders before membership expires |
| 💳 **Razorpay Payments** | India's #1 payment gateway — UPI, cards, net banking |
| 🏋️ **Workout Plans** | Create & assign personalized programs per member |
| 🥗 **Diet Plans** | Nutrition assignments with macros and meal schedules |
| 🏢 **Multi-branch** | Manage all gym locations from one admin account |
| 👨‍💼 **Staff Management** | Role-based access for receptionists, trainers |
| 📊 **Analytics** | Revenue trends, member growth, attendance heatmaps |
| ⏰ **Fees & Expiry** | Instant view of expiring memberships, bulk renewals |
